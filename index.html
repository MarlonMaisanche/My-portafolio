Desarrollos en Flutter

En Flutter se pueden desarrollar aplicaciones multiplataforma, este sdk nos ayuda a reducir los tiempos de desarrollo ya que facilita en gran medida el desarrollo de aplicaciones ya que no hay necesidad de crear nuevo código para cada plataforma.

Desarrollos en TypeScript

TypeScript es un lenguaje de programación libre y de código abierto desarrollado y mantenido por Microsoft. Es un superconjunto de JavaScript, que esencialmente añade tipos estáticos y objetos basados en clases.

Desarrollos en .Net

.NET es un framework de Microsoft que hace un énfasis en la transparencia de redes, con independencia de plataforma de hardware y que permite un rápido desarrollo de aplicaciones.

Desarrollos en Java

Java es un lenguaje de programación y una plataforma informática que fue comercializada por primera vez en 1995 por Sun Microsystems. Hay muchas aplicaciones y sitios web que no funcionarán, probablemente, a menos que tengan Java instalado, y cada día se crean más. Java es rápido, seguro y fiable.
